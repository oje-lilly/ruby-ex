public class MonitoringSessionImpl
extends java.lang.Object
implements MonitoringSession

The MonitoringSessionImpl class provides a DRMAA interface to Grid Engine. This interface is built on top of the DRMAA C binding using JNI. In order to keep the native code as localized as possible, this class also provides native DRMAA services to other classes, such as the JobTemplateImpl.

This class relies on the version 1.0 drmaa2 shared library.
