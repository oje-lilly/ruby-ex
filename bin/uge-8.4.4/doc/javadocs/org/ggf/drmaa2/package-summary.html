Package org.ggf.drmaa2 Description

Provides an implementation of the Distributed Resource Management Application API (DRMAA2). DRMAA2 is used for interacting with a Distributed Resource Manager (DRM). The Session interface is the main point of interaction and provides the ability to submit, monitor, and control jobs.
